# Version 3

The third major iteration of my custom robotic arm.

This version focused on refining the mechanical design while significantly improving the robot's software architecture. The arm now features a stronger dual-support shoulder design, improved actuator mounting, and coordinated multi-axis motion using the AccelStepper library, creating a solid foundation for future kinematics, URDF generation, and ROS2 integration.

## Folder Contents

📷 **Images** – CAD renders, assembly images, and progress photos.

💻 **Code** – Arduino firmware implementing simultaneous multi-axis motion, software homing, and predefined robot poses.

📄 **Design Review** – Engineering review documenting the mechanical and software improvements made over Version 2.
