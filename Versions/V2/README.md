# Version 2

The second major iteration of my custom robotic arm.

This version focused on improving the mechanical design while introducing software homing, laying the foundation for more advanced robotics concepts.

## Folder Contents

- 📷 **Images** – CAD renders and progress photos.
- 💻 **Code** – Software homing program and supporting Arduino code.
- 📄 **Design Review** – Engineering review documenting improvements over Version 1.
